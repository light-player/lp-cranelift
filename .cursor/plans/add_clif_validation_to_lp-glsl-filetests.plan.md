# Add CLIF Validation to lp-glsl-filetests

## Problem

The CLIF IR generated by lp-glsl may be invalid (e.g., undefined stack slot `ss0`), but lp-glsl-filetests doesn't validate it before running tests. This allows invalid CLIF to reach the backend, causing confusing errors. Cranelift filetests validates CLIF using `verify_function()` before running tests.

## Solution

Add CLIF validation at three points in lp-glsl-filetests, matching cranelift filetests:

1. **After compilation** (test_compile.rs) - validate CLIF before transformations
2. **After transformation** (test_transform.rs) - validate CLIF after fixed32 transformation  
3. **Before execution** (test_run/mod.rs) - ensure validation happens before running code

## Implementation

### 1. Create validation helper function

Create a new helper module `lightplayer/crates/lp-glsl-filetests/src/validation.rs`:

```rust
use anyhow::{Context, Result};
use cranelift_codegen::isa::TargetIsa;
use cranelift_codegen::print_errors::pretty_verifier_error;
use cranelift_codegen::settings::FlagsOrIsa;
use cranelift_codegen::verify_function;
use lp_glsl::ClifModule;

/// Validate all functions in a ClifModule
pub fn validate_clif_module(module: &ClifModule, isa: &dyn TargetIsa) -> Result<()> {
    // Validate user functions
    for (_, func) in module.user_functions() {
        verify_function(func, FlagsOrIsa { flags: isa.flags(), isa: Some(isa) })
            .map_err(|errors| {
                anyhow::anyhow!(
                    "CLIF validation failed for function '{}':\n{}",
                    func.name,
                    pretty_verifier_error(func, None, errors)
                )
            })
            .with_context(|| format!("failed to validate function '{}'", func.name))?;
    }

    // Validate main function
    verify_function(module.main_function(), FlagsOrIsa { flags: isa.flags(), isa: Some(isa) })
        .map_err(|errors| {
            anyhow::anyhow!(
                "CLIF validation failed for main function:\n{}",
                pretty_verifier_error(module.main_function(), None, errors)
            )
        })
        .with_context(|| "failed to validate main function")?;

    Ok(())
}
```

### 2. Add validation to test_compile.rs

In `lightplayer/crates/lp-glsl-filetests/src/test_compile.rs`:

- Import the validation module
- After compiling to CLIF (line 80), call `validate_clif_module(&module, isa.as_ref())?` before formatting/comparing
