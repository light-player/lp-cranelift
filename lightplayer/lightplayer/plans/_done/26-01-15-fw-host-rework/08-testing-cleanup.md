# Phase 8: Testing and cleanup

## Goal

Final cleanup phase: remove temporary code, fix warnings, ensure tests pass, format code.

## Tasks

1. Remove any temporary code, TODOs, debug prints
2. Fix all warnings
3. Ensure all tests pass
4. Ensure code is clean and readable
5. Run `cargo +nightly fmt` on lightplayer/ directory
6. Move plan directory to `lightplayer/plans/_done/`

## Success Criteria

- No warnings
- All tests pass
- Code is formatted
- Plan is moved to `_done/`
- Ready to commit
